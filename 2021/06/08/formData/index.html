发表于 2021-06-08 更新于 2022-09-26
本文字数： 3.5k 阅读时长 ≈ 3 分钟

FormData的主要用途有两个

  1. 将form表单元素的name与value进行组合，实现表单数据的序列化，从而减少表单元素的拼接，提高工作效率。
  2. 异步上传文件

  2. 通过表单初始化formData

操作方法

  1. 通过get(key)与getAll(key)来获取相对应的值
  2. 通过append(key,value)在数据末尾追加数据
  3. 通过set(key, value)来设置修改数据
  4. key的值存在，会修改对应的value值
  5. 通过has(key)来判断是否存在对应的key值
  6. 通过delete(key)可以删除数据
