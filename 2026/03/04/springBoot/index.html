Spring核心注解

  1. 把类交给 Spring 管理 │ @Component │ 通用组件 │ 工具类、处理器 │ │ @Configuration │ 配置类 │ Config 配置 │ │ @Service │ 业务逻辑层 │ UserService │ │ @Repository │ 数据访问层 │ Mapper/DAO │ │ @Controller │ 返回页面（老式） │ 很少用 │ │ @RestController │ 返回 JSON 数据 │ 最常用 │
  2. 依赖注入（获取 Spring 管理的对象）
@Service
public class UserService {
    @Autowired //自动注入
    private UserMapper userMapper;  // Spring自动把userMapper注入进来
}

│ @Resource │ 自动注入（按名称） │

Web 层

@RestController等于@Controller + @ResponseBody(返回JSON) @RequestMapping(“/api/user”)整个类的基础路径 @GetMapping(“/list”)等价于@RequestMapping(value=”/list”, method=GET)

参数绑定

  1. @PathVariable │ 路径变量 │ /user/{id}
/user/123
@GetMapping("/{id}")
  1. @RequestParam │ URL 参数 │ ?name=张三
/search?name=张三&page=1
@GetMapping("/search")
public List<User> search(@RequestParam String name, @RequestParam Integer page) { }
  1. @RequestBody │ JSON 请求体 │ POST 的 body │
POST /user  Body: {"name": "张三"}
@PostMapping
public void add(@RequestBody User user) { }

数据访问(MyBatis)

@Mapper  // 告诉MyBatis这是一个Mapper接口
public interface UserMapper {
  @Select("SELECT * FROM sys_user WHERE id = #{id}")
  User selectById(Long id);
  @Insert("INSERT INTO sys_user(name) VALUES(#{name})")
  int insert(User user);
}

数据访问( MyBatis-Plus)

@Data
@TableName("sys_user")  // 对应数据库表名
public class User {
  @TableId(type = IdType.AUTO)  // 主键自增
  private Long id;
  private String username;
  @TableField(exist = false)  // 这个字段不在数据库中
  private String tempField;
}

参数校验

  1. 添加了@Valid注解后会自动校验实体类中加了注解的字段 @NotNull不能为 null @NotEmpty不能为 null 且长度/大小 > 0 @NotBlank字符串不能为 null 且去空格后长度 > 0 @Size(min, max)字符串/集合/数组长度范围@Min / @Max数值最小/最大值 @Email邮箱格式 @Pattern(regexp)正则匹配
@PostMapping("/add")
public AjaxResult add(@Valid @RequestBody User user) {
    // @Valid 触发校验
}
@Data
public class User {
    @NotBlank(message = "用户名不能为空")
    @Size(min = 2, max = 20, message = "用户名长度2-20个字符")
    private String username;
    @Email(message = "邮箱格式不正确")
    private String email;
    @NotNull(message = "年龄不能为空")
    private Integer age;
}
  1. 嵌套校验
// 如果对象中还有对象，需要在字段上也加 @Valid：
public class Order {
    @Valid  // 触发 User 对象的校验
    private User user;

    @NotNull
    private BigDecimal amount;
}

Spring Boot 有非常多的知识点，我按模块给你梳理一下：

  1. 核心基础

自动配置原理：@SpringBootApplication、@EnableAutoConfiguration、条件注解（@ConditionalOnClass 等）配置文件：application.yml/properties、多环境配置（dev/test/prod）、配置优先级依赖注入：@Autowired、@Resource、构造器注入 Bean 生命周期：@PostConstruct、@PreDestroy、InitializingBean

  1. Web 开发

RESTful API：@RestController、@RequestMapping、@PathVariable、@RequestParam 参数校验：@Valid、@Validated、自定义校验器全局异常处理：@ControllerAdvice、@ExceptionHandler 拦截器 vs 过滤器：HandlerInterceptor、Filter 跨域处理：@CrossOrigin、CORS 配置文件上传下载：MultipartFile

  1. 数据库相关

MyBatis/MyBatis-Plus：XML 映射、注解开发、分页插件、代码生成器 JPA：@Entity、JpaRepository、JPQL 事务管理：@Transactional、事务传播机制、隔离级别数据源配置：Druid、HikariCP 多数据源：动态切换数据源

  1. 安全认证

Spring Security：认证与授权、密码加密、Remember Me JWT：Token 生成与验证、刷新 Token OAuth 2.0：第三方登录权限控制：RBAC 模型、@PreAuthorize

  1. 缓存技术

Spring Cache：@Cacheable、@CacheEvict、@CachePut Redis：RedisTemplate、序列化配置、分布式锁本地缓存：Caffeine、Guava Cache

  1. 消息队列

RabbitMQ：交换机类型、消息确认、死信队列Kafka：生产者与消费者、分区与副本RocketMQ：顺序消息、事务消息

  1. 任务调度

定时任务：@Scheduled、Cron 表达式异步任务：@Async、线程池配置分布式任务：XXL-Job、Quartz

  1. 日志管理

Logback/Log4j2：日志级别、自定义格式MDC：分布式链路追踪ELK：日志收集与分析

  1. 测试

单元测试：@SpringBootTest、@MockBean接口测试：MockMvc、Postman集成测试：TestContainers

  1. 微服务相关

Spring Cloud：Nacos、Gateway、OpenFeign、Sentinel服务注册与发现：Eureka、Consul配置中心：Apollo、Spring Cloud Config限流熔断：Hystrix、Resilience4j

  1. 性能优化

线程池：ThreadPoolExecutor、@Async 线程池连接池：数据库连接池、HTTP 连接池 JVM 调优：GC 日志分析、内存溢出排查慢 SQL 优化：索引、执行计划

  1. 监控运维

Actuator：健康检查、指标监控 Prometheus + Grafana：可视化监控 Skywalking/Zipkin：链路追踪 Docker 部署：Dockerfile、容器化

  1. 其他实用功能

AOP：切面编程、自定义注解事件机制：ApplicationEvent、@EventListener WebSocket：实时通信邮件发送：JavaMailSender Excel 导入导出：EasyExcel、POI 接口文档：Swagger/Knife4j、Apifox
