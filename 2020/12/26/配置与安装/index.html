git-ssh配置

git config --global user.name "zhangjie"
git config --global user.email "2811932734@qq.com"
ssh-keygen -t rsa -C "2811932734@qq.com"

常用命令

git branch    查看当前分支
git branch -r   查看远程分
git branch -a   查看本地分支和远程分支
git checkout feature/zhangjie   切换分支（feature/zhangjie）
git branch --set-upstream-to=origin/zj zj  本地分支和远程建立联系
git branch -D  分支名  删除本地分支

合并分支

  1. 将sp2上修改的内容合并到sp3上
git checkout sP3  //先切换到目标分支上
git merge SP2  //执行合并操作
  1. 无冲突执行提交
git push
